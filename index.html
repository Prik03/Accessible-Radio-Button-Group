Accessible Radio Buttons Example

Grouping

Grouping radio buttons using a <fieldset> and <legend> helps users understand that the options belong to the same question or category. It also improves accessibility — screen readers can announce the group label before reading each option, giving users better context about what they are choosing.

Importance of the name Attribute

The name attribute groups radio buttons together, ensuring that only one option can be selected at a time within the same group. It also improves accessibility — screen readers can better identify the relationship between radio buttons and provide smoother keyboard navigation.

Label Association

Associating a label with an input helps users — especially those using screen readers or assistive technologies — understand what each control does. Clicking the label also activates the input, improving usability.
